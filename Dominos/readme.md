# Domino's Website Clone

## Project Overview

This project is a front-end website inspired by the Domino's Pizza website. It was developed using HTML and CSS as part of my web development learning journey.

## Features

* Home Page
* Menu Page
* Contact Page
* Login Page
* Registration Page
* Responsive and user-friendly design
* Navigation bar for easy page access

## Technologies Used

* HTML5
* CSS3

## Purpose

The main objective of this project was to improve my front-end web development skills by creating a multi-page website with an attractive layout and proper navigation.

## Learning Outcomes

* Understanding HTML page structure
* Styling web pages using CSS
* Creating navigation menus
* Designing responsive layouts
* Building forms for login and registration
